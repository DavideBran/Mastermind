Mastermind

Mastermind is a game where one player, the Codemaker, creates a secret code of four distinct symbols from a set of eight possible colors: red, orange, grey, white, purple, green, pink, and blue. The other player, the Codebreaker, has nine turns to guess the code. The Codebreaker selects a sequence of four unique symbols, trying to match the hidden code. After each guess, the Codemaker provides feedback in the form of colored pegs. A black peg indicates a correct symbol in the correct position, while a orange peg signifies a correct symbol but in the wrong position. The game continues with the Codebreaker using deduction and logic to refine their guesses based on the feedback received. If the Codebreaker successfully deciphers the sequence within nine turns, they win. However, if you're the Codemaker and the Codebreaker doesn't solve the code within six moves, you win. Remember, no duplicate symbols are allowed in the Codebreaker's guesses, and the key to winning is using strategy and deduction to crack the hidden code within the given number of attempts!
